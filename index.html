Composant Header ShootMe

Voici un composant header réutilisable, responsive et accessible.

Comment l'utiliser

Ajoutez simplement ce code à votre HTML :

<script src="components/header-component.js" defer></script> <shoot-me-header active-page="home"></shoot-me-header>

Options

Vous pouvez définir la page active en modifiant l'attribut active-page :

  • home - Page d'accueil
  • portfolio - Page portfolio
  • about - Page à propos
  • contact - Page contact
